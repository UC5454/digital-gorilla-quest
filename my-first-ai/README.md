# 🦍 My First AI Employee

ここにあなたのAI社員を作ります！

## 概要

エリア6（祭壇）以降のクエストで、このフォルダを使ってAI社員を構築していきます。

## フォルダ構成

```
my-first-ai/
├── README.md           # このファイル
├── members.yaml        # AI社員の名簿（エリア7で作成）
├── CLAUDE.md           # AI社員の設計図（エリア6で作成）
├── tools/
│   └── start-employee.sh  # AI社員の起動スクリプト（エリア7で作成）
└── docs/
    └── README.md       # 活用シーン設計や振り返りメモ
```

## 進め方

1. **エリア6（祭壇）**: AI社員のCLAUDE.mdを設計・作成
2. **エリア7（砦）**: チーム連携の仕組み（members.yaml、INBOX.md、start-employee.sh）を構築
3. **エリア8（火山）**: Agent Teamsで並列処理を体験
4. **エリア9（頂上）**: 全てを統合して完成！

## 注意

- 困ったら `/go hint` で先輩ゴリラに聞こう
