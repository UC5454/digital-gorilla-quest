# /status - ステータス確認

ゴリラの詳細ステータスを表示する。

## 起動時の動作

1. `gorilla_data/progress.json` を読み込む
2. 各種データを集計
3. ステータス画面を表示

## ゴリラ画像表示

ステータス表示時、まず現在の進化段階のゴリラ画像をiTerm2インラインで表示する：
```bash
bash tools/show-gorilla.sh auto 40
```

## パワーゲージの計算

現在のレベルから次のレベルまでの進捗率を20文字のゲージで表現。
`level_thresholds` を参照。

```
進捗率 = (現在BP - 現在レベル閾値) / (次レベル閾値 - 現在レベル閾値) * 100
ゲージ = █ × (進捗率/5) + ░ × (20 - 進捗率/5)
```
