# /jungle - ジャングルマスター

ゲーム進行管理とナビゲーションを行うメインスキル。

## 起動時の動作

1. `gorilla_data/progress.json` を読み込む
2. 初回起動の場合は名前入力を促し、created_atを設定
3. 現在の状況に応じた案内を表示

## 表示内容

```
🦍🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🦍

  ウホホ！！！

  デジタルジャングルへようこそ！

  我は「ジャングルマスター」
  この広大なデジタルの密林を統べる者だ。

  お前も最強のゴリラを目指すのか？
  ならば名を教えよ！

🦍🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🌴🦍
```

名前を聞いた後、progress.jsonを更新し、ゲーム説明を表示。

エリアマップの凡例：
- 📍 = 現在のエリア
- ✅ = ボス撃破済み
- 🔒 = 未解放

### エリア制覇時
```
🦍💥 ボス戦解禁！💥🦍

  ウホホホ！！
  [エリア名]の全クエストを制覇したな！

  ボス「[ボス名]」が縄張りを守っている！

  /go boss で挑戦せよ！
```

## 状態判定ロジック

1. 初回かどうか（created_atがnull）
2. 現在のエリアのクエスト進捗
3. ボス戦が可能か
4. 次のクエスト番号の特定

## progress.json更新

- last_played を現在時刻に更新
- 初回の場合は名前とcreated_atを設定
