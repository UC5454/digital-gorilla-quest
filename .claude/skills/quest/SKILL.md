# /quest - クエスト開始

指定されたクエストの内容を表示し、冒険を開始する。

## 使用方法

```
/quest [番号]
```

番号を省略した場合は、現在のエリアで次にクリアすべきクエストを表示。

## 起動時の動作

1. `gorilla_data/progress.json` を読み込む
2. クエスト番号に対応するクエストファイルを読み込む
3. クエスト内容を演出付きで表示
4. progress.jsonのcurrent questを更新

## クエストファイルの場所

| クエスト番号 | ファイルパス |
|-------------|-------------|
| Q1-Q6 | quests/grassland/q01_xxx.md ~ q06_xxx.md |
| Q7-Q12 | quests/forest/q07_xxx.md ~ q12_xxx.md |
| Q13-Q18 | quests/mountain/q13_xxx.md ~ q18_xxx.md |
| Q19-Q24 | quests/cave/q19_xxx.md ~ q24_xxx.md |
| Q25-Q30 | quests/summit/q25_xxx.md ~ q30_xxx.md |

## 表示フォーマット

```
🦍🌴 QUEST [番号] 🌴🦍
================================

  📍 [エリア名] - [クエスト名]

  [クエストの導入テキスト（ゴリラ口調）]

  【目標】
  [目標リスト]

  【成功条件】
  [条件リスト]

  【報酬】
  🍌 [BP] BP
  [バッジがある場合: 🥇 バッジ「XXX」]

================================
  💡 困ったら /hint で先輩ゴリラに聞け！
  ✅ 完了したら /banana で報告！
```

## バリデーション

- 未解放エリアのクエストは開始不可
- 前のクエストが未完了の場合は警告を表示（ただし開始は可能）
- 既にクリア済みのクエストは再挑戦として表示
