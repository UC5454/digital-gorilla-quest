Quest 2-3: Git初期化マスター

Gitリポジトリを初期化し、初めてのコミットを行おう

📋 コピペセクション

以下のテキストをClaude Codeにコピペして実行しよう

Step 1: リポジトリ初期化 
新しいプロジェクトフォルダ「my-first-project」を作成して、gitリポジトリとして初期化してください
Step 2: README作成 
README.mdを作成して、プロジェクト名と説明を書いてください。その後git addでステージングしてください
Step 3: 初コミット 
ステージングした変更をコミットしてください。コミットメッセージは「Initial commit: プロジェクト開始」にしてください

💡 ヒント

git initはフォルダをGitリポジトリに変換するコマンドです。これにより、そのフォルダ内のファイル変更を追跡できるようになります。Claude Codeに「git initして」と言うだけでOKです。

ステージング（git add）は「次のコミットに含める変更を選ぶ」こと。コミット（git commit）は「選んだ変更を確定する」ことです。お買い物カートに入れる → レジで購入、のイメージです。

Claude Codeに「git statusを見せて」と言ってみましょう。現在の状態が分かります。エラーが出たらそのままClaude Codeに「このエラーを解決して」と伝えてOKです。

🎯 成功条件

  • 1 .gitフォルダが作成されている
  • 2 README.mdが存在する
  • 3 git logで最低1つのコミットが確認できる
  • 4 git statusで「nothing to commit」と表示される
