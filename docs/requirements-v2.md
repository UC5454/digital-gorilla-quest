## クエストファイル形式（テンプレート）

各クエストファイルは以下の形式で作成する:

```markdown
# Quest [番号]: [クエスト名]

## 🦍 導入

[ゴリラ口調のストーリーテキスト。3-5行。親しみやすく、学習意欲を高める内容]

## 🎯 目標

- [ ] [目標1]
- [ ] [目標2]
- [ ] [目標3]

## 📋 やること

[具体的な手順を番号付きで記載]

## 📎 コピペ

```
[そのまま使えるコマンドやコード]
```

## ✅ 成功条件

- [ ] [条件1]
- [ ] [条件2]
- [ ] [条件3]

## 📦 成果物

- [ファイルパスや成果物の説明]

## 🍌 報酬

- base_bp: [数値]
- first_clear_bonus: [数値]
- badge: [バッジ名]（該当クエストのみ）
- badge_icon: [絵文字]
- badge_description: [説明]

## 💡 ヒント

### hint_1
[軽いアドバイス。方向性を示す]

### hint_2
[具体的なヒント。コマンドの一部や手順の一部を提示]

### hint_3
[ほぼ答え。直接的な解答に近い情報]

## 📚 学習ポイント

- [重要な学び1]
- [重要な学び2]
- [重要な学び3]

## 📖 関連リファレンス

docs/reference/[XX]_[name].md
```

## エリア別クエスト仕様

### エリア1: 草原（grassland）Q1-Q6
- Q1: 最初のウホウホ会話（5分）— 自由対話、基本操作確認
- Q2: ファイルを読む力（10分）— README.mdとCLAUDE.mdをReadで読む
- Q3: ファイルを創る力（10分）— practice/grassland/self-intro.md をWriteで作成
- Q4: ファイルを編集する力（10分）— self-intro.mdをEditで編集
- Q5: ジャングルの声を聴く（10分）— Bashでシェルコマンド実行
- Q6: ツールの組み合わせ（10分）— 4ツール複合タスク
- Boss: 門番ゴリラ（10分）— 4ツール組み合わせ実践

### エリア2: 森林（forest）Q7-Q12
- Q7: 記憶の大樹を発見せよ（10分）— CLAUDE.md解析
- Q8: 最初の記憶を刻む（10分）— CLAUDE.md新規作成
- Q9: ルールの力（10分）— コーディングルール設定と検証
- Q10: 禁止の力（10分）— 禁止事項の設定と検証
- Q11: 記憶の継承（10分）— 3層構造、CLAUDE.local.md
- Q12: 記憶の設計術（10分）— ベストプラクティス
- Boss: 賢者ゴリラ（10分）— オリジナルCLAUDE.md作成

### エリア3: 山岳（mountain）Q13-Q18
- Q13: 道場への入門（10分）— Skills構造理解

- Boss: 武闘家ゴリラ（10分）— オリジナルワークフロースキル

### エリア4: 洞窟（cave）Q19-Q24

- Q24: 3つのハンドラー（10分）— command/prompt/agent
- Boss: 魔術師ゴリラ（10分）— 実用Hook設定

### エリア5: 神殿（temple）Q25-Q30
- Q25: 神殿の門をくぐる（10分）— MCP概念理解

- Q29: MCPワークフロー（20分）— 複合ワークフロー

### エリア6: 祭壇（altar）Q31-Q36
- Q31: AI社員とは何か（10分）— チャットボット vs AI社員

- Q33: 役割の設計 role（15分）— 責任範囲/判断基準

- Q35: スキルの設計 skills（15分）— 得意/不得意/ワークフロー
- Q36: 記憶の設計 startup & memory_actions（20分）— 起動/記録/確認ルール
- Boss: 霊媒ゴリラ（15分）— AI社員CLAUDE.md完成テスト

### エリア7: 砦（fortress）Q37-Q42
- Q37: 群れの掟（10分）— members.yaml作成

- Q43: 炎の覚醒（10分）— Agent Teams概念

- Boss: 炎龍ゴリラ（10分）— 3エージェント混合ワークフロー

### エリア9: 頂上（summit）Q49-Q54
- Q49: 王の構想（10分）— 活用シーン設計
- Q50: 設計図の完成（10分）— CLAUDE.md最終調整
