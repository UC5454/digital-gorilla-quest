# 🌴 デジタルジャングルの世界設定 🌴

## 世界観

### デジタルジャングルとは

広大な仮想世界「デジタルジャングル」。
ここでは、コードを操るゴリラたちが日々鍛錬を積んでいる。

株式会社デジタルゴリラの社員たちは、このジャングルで修行を積み、
最強の「シルバーバック」、そして伝説の「ジャングルの王」を目指す。

### なぜゴリラなのか

ゴリラは強さと知性を兼ね備えた動物。
コードを書く開発者も同様に、パワー（実装力）と知恵（設計力）が必要だ。

そして何より、ゴリラは群れで協力し合う。
チーム開発の精神を象徴する存在なのだ。

## 5つのテリトリー

### 🌿 草原エリア - はじまりの地

最初に足を踏み入れる場所。
見渡す限りの緑の草原が広がり、優しい風が吹いている。

ここでは Claude Code の基本を学ぶ。
インストール、起動、ファイル操作など、全ての基礎がここにある。

**ボス: 門番ゴリラ**
草原の入り口を守る厳格なゴリラ。
基本ができていない者は先に進めない。

### 🌲 森林エリア - 記憶の森

深い森の中、巨大な木々がそびえ立つ。
この森には「記憶の大樹」と呼ばれる伝説の木がある。

ここでは CLAUDE.md の使い方を学ぶ。
プロジェクトの記憶を管理する方法を身につける。

**ボス: 賢者ゴリラ**
記憶の大樹に住む知恵深いゴリラ。
過去の知識を正しく継承できる者だけを認める。

### ⛰️ 山岳エリア - 技の峰

険しい山道が続く修行の地。
頂上にはスキルを極めた達人たちの道場がある。

ここでは Skills の作成を学ぶ。
自分だけの技（スキル）を編み出す力を身につける。

**ボス: 武闘家ゴリラ**
山頂の道場を守る武術の達人。
オリジナルの技を持たない者は弟子として認めない。

### 🕳️ 洞窟エリア - 連鎖の洞窟

地下深くに広がる神秘的な洞窟。
光る結晶が連鎖反応を起こし、美しく輝いている。

ここでは Hooks の設定を学ぶ。
アクションの連鎖を制御する魔法を習得する。

**ボス: 魔術師ゴリラ**
洞窟の奥深くに住む神秘的なゴリラ。
連鎖の魔法を使いこなせない者は通さない。

### 👑 頂上エリア - 王の座

全てのエリアを制覇した者だけが到達できる聖地。
ここには伝説のキングコングが待っている。

全ての知識を統合し、ベストプラクティスを実践する。
真のジャングルの王としての実力を証明せよ。

**ボス: キングコング**
全てを極めし伝説のゴリラ。
真の実力者のみを王として認める。

## ゴリラの進化

### 進化の段階

1. **🐵 子ゴリラ** (Lv.1-5)
   まだ小さくてかわいい見習いゴリラ。
   好奇心旺盛で、何でも学ぼうとする。

2. **🦧 若ゴリラ** (Lv.6-15)
   少し筋肉がついてきた若者ゴリラ。
   基本的なことは理解し始めている。

3. **🦍 成熟ゴリラ** (Lv.16-25)
   立派な体つきになった一人前のゴリラ。
   仲間からも認められる存在。

4. **🦍💪 ブラックバック** (Lv.26-35)
   黒い毛並みが美しく輝くゴリラ。
   群れの中でリーダー候補として期待される。

5. **🦍✨ シルバーバック** (Lv.36-45)
   銀色の背中が威厳を放つ群れのリーダー。
   若いゴリラたちの手本となる存在。

6. **👑🦍 ジャングルの王** (Lv.46-50)
   全てを極めた伝説のゴリラ。
   デジタルジャングル全体から尊敬される存在。

## バナナポイント（BP）

ゴリラのパワーの源。
クエストをクリアすると獲得でき、一定量貯まるとレベルアップする。

### BP獲得方法
- クエストクリア: 100-300 BP
- 初回クリアボーナス: +20%
- ボス撃破: 500-1000 BP
- 特別バッジ獲得時: ボーナス BP

## ジャングルマスター

デジタルジャングルを統べる謎の存在。
プレイヤーを導き、冒険の案内をしてくれる。

口癖は「ウホッ！」「ウホホ！」
厳しくも優しい、頼れるガイド役。

## 先輩ゴリラ

困ったときに助けてくれる親切な先輩。
`/hint` を呼ぶと現れて、アドバイスをくれる。

かつて自分も同じ道を歩んできたので、
初心者の気持ちがよく分かる。
