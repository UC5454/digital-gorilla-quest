# 🦍 修行場（Practice Area）

ここは Claude Code の練習を行う場所です。

## フォルダ構成

```
practice/
├── skill-dojo/      # スキル道場 - スキル作成の練習（エリア3: 山岳）
├── hook-cave/       # フック洞窟 - フック設定の練習（エリア4: 洞窟）
├── memory-tree/     # 記憶の大樹 - CLAUDE.md の練習（エリア2: 森林）
├── boss-challenge/  # ボス戦チャレンジ - ボス戦用の作業フォルダ
├── forest/          # 森林エリア追加練習
├── mountain/        # 山岳エリア追加練習
├── cave/            # 洞窟エリア追加練習
└── temple/          # 神殿エリア - MCP設定の練習（エリア5）
```

## 使い方

各フォルダで自由に練習してください。
失敗しても大丈夫。ここは修行の場です！

### スキル道場（skill-dojo）

スキルの作成を練習する場所。
`.claude/skills/[name]/SKILL.md` の形式で作成してください。

### フック洞窟（hook-cave）

フックの設定を練習する場所。
`.claude/settings.json` を作成してください。

### 記憶の大樹（memory-tree）

CLAUDE.md の作成を練習する場所。
プロジェクト設定の書き方を学びましょう。

### 神殿（temple）

MCP設定の練習場所。
`.mcp.json` の書き方を学びましょう。

### ボス戦チャレンジ（boss-challenge）

ボス戦で出される課題の作業フォルダ。
各エリアのボス戦成果物をここに保存できます。

## 注意

- このフォルダ内のファイルは自由に作成・削除してOK
- 失敗を恐れずにどんどん試してみよう
- 困ったら `/go hint` で先輩ゴリラに聞こう
