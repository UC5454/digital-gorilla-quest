# 🦍 ボスチャレンジ用フォルダ

ここは各エリアのボス戦で使う作業フォルダです。

## フォルダ構成

```
boss-challenge/
├── sage-test/           # 賢者ゴリラ戦用（CLAUDE.md作成）
├── warrior-test/        # 武闘家ゴリラ戦用（スキル作成）
│   └── .claude/skills/
├── mage-test/           # 魔術師ゴリラ戦用（フック設定）
│   └── .claude/
└── final-project/       # キングコング戦用（最終プロジェクト）
    └── .claude/skills/
```

## 使い方

各ボス戦で指定されたタスクをこのフォルダ内で実施してください。

ボス戦の詳細は `/boss` コマンドで確認できます。
