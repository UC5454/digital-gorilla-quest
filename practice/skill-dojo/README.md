# 🥋 スキル道場

ここはスキル作成を練習する場所です。

## 構造

```
skill-dojo/
└── .claude/
    └── skills/
        └── [your-skill-name]/
            └── SKILL.md
```

## スキルの作り方

1. `.claude/skills/` の下に、スキル名のフォルダを作成
2. その中に `SKILL.md` を作成
3. `/your-skill-name` で呼び出せる！

## SKILL.md テンプレート

```markdown
# /skill-name - スキルの説明

詳細な説明をここに。

## 起動時の動作

1. 最初にやること
2. 次にやること
3. 結果を表示

## 出力形式

期待される出力の形式
```

ウホッ！自分だけの技を編み出せ！ 🦍
