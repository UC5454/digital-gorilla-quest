```
memory-tree/

```

## CLAUDE.md テンプレート

```markdown
# プロジェクト名

## 概要
このプロジェクトの説明

## 技術スタック
- 言語: JavaScript
- フレームワーク: Node.js
- その他: npm

## コーディングルール
- インデント: スペース2つ
- 命名規則: camelCase
- コメント: 日本語OK

## 禁止事項
- console.log を本番コードに残さない
- any型を使わない

## コマンド
- `npm run dev` - 開発サーバー起動
- `npm test` - テスト実行
```

## ポイント

1. **簡潔に**: 長すぎない説明
2. **具体的に**: 曖昧な表現を避ける
3. **更新する**: プロジェクトの変化に合わせて
