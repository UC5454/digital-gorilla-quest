# Quest 10: 掟を定める

## 導入テキスト

ウホッ！個人設定も作れるようになったな。

次は「掟」を定めるときだ。
プロジェクトには、守るべきルールが必要だ。

CLAUDE.md に明確なルールを追加して、Claude Code に従わせてみろ。

## 目標

- CLAUDE.md にプロジェクトルールを追加する
- そのルールが実際に適用されることを確認する

## 成功条件

- [ ] practice/memory-tree/CLAUDE.md に「禁止事項」または「ルール」セクションを追加
- [ ] Claude Code がそのルールに従った動作をすることを確認

## 報酬

- base_bp: 400
- first_clear_bonus: 80

## ヒント

### hint_1
例えば「console.log を使わない」「英語のコメントは禁止」
などのルールを設定してみろ。

### hint_2
ルールを追加したら、Claude Code に
「このプロジェクトのルールを教えて」と聞いてみろ。
正しく認識しているか確認だ。

### hint_3
practice/memory-tree/CLAUDE.md を編集して、以下を追加：

## 禁止事項
- console.log は使用禁止（代わりに専用ロガーを使う）
- any 型は使用禁止
- マジックナンバーは禁止（定数を使う）

## 必須事項
- 関数には必ずコメントをつける
- 変数名は日本語でも可

その後、「このフォルダでJavaScriptファイルを作って」と頼んで、
ルールに従うか確認しろ！
