# Quest 9: 秘密の記憶

## 導入テキスト

ウホホ！自分の CLAUDE.md を作れたな！

だが、全ての記憶を共有する必要はない。
時には、自分だけの秘密の記憶も必要だ。

CLAUDE.local.md という、個人用の設定ファイルについて学ぼう。

## 目標

- CLAUDE.local.md の役割を理解する
- プロジェクトルートに CLAUDE.local.md を作成する

## 成功条件

- [ ] CLAUDE.local.md が個人設定用であることを理解した
- [ ] CLAUDE.local.md を作成した

## 報酬

- base_bp: 300
- first_clear_bonus: 60

## ヒント

### hint_1
CLAUDE.local.md は .gitignore に入れるべきファイルだ。
なぜなら、個人的な設定やパスが含まれることがあるからな。

### hint_2
例えばこんな内容を入れる：
- 自分の名前や好みの設定
- ローカル環境固有のパス
- 個人的なメモ

### hint_3
Claude Code に「CLAUDE.local.md を作成して。内容は：
# 個人設定

## 私の情報
- 名前: [あなたの名前]
- 好みのエディタ: VS Code

## メモ
- 今日から Claude Code の勉強を開始！
」と頼んでみろ！
