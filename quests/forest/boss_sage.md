## ボスチャレンジ

### 課題内容

CLAUDE.md の真の力を証明せよ！

以下のタスクを完了せよ：

### タスク1: 完璧な CLAUDE.md を作成

`practice/boss-challenge/sage-test/` フォルダに、以下の要素を全て含む CLAUDE.md を作成せよ：

必須セクション：
1. **プロジェクト概要** - 何のプロジェクトか
2. **技術スタック** - 使用技術（最低3つ）
3. **コーディングルール** - 守るべきルール（最低3つ）
4. **禁止事項** - やってはいけないこと（最低2つ）
5. **コマンド一覧** - よく使うコマンド（最低3つ）

### タスク2: ルールの適用確認

作成した CLAUDE.md があるフォルダで Claude Code を起動し、
「このプロジェクトのルールを教えて」と聞いて、
正しくルールが認識されていることを確認せよ。

### タスク3: ルールに従ったコード生成

Claude Code に「このプロジェクトのルールに従って、サンプルコードを作成して」と依頼し、
CLAUDE.md で定義したルールが実際に適用されることを確認せよ。

## 成功条件

- [ ] `practice/boss-challenge/sage-test/CLAUDE.md` が存在する
- [ ] 5つの必須セクションが全て含まれている
- [ ] Claude Code がルールを正しく認識している
- [ ] 生成されたコードがルールに従っている
