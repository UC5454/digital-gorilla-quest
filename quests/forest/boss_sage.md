## ⚔️ チャレンジ

オリジナルのAI社員のCLAUDE.mdを一から設計・作成せよ！

### 1. AI社員の企画
自分だけのオリジナルAI社員を考えろ。仕事用でも趣味用でもいい。

### 2. CLAUDE.md作成
`practice/boss-challenge/forest/my-ai-employee/CLAUDE.md` を作成し、以下の全セクションを含めろ。

```
practice/boss-challenge/forest/my-ai-employee/CLAUDE.md を作成して。
自分のオリジナルAI社員を設計して以下のセクション全てを含めること：

1. identity（名前、役割、一人称）
2. role（責任範囲3つ以上、判断基準）
3. persona（性格、話し方の特徴、NGワード）
4. rules（やるべきこと3つ以上、禁止事項3つ以上）
5. skills（得意なこと3つ以上）

テーマは自由。例: 料理アシスタント、英語学習コーチ、ゲーム実況AI、日記サポーター等
```

### 3. 設計レビュー
作成したCLAUDE.mdをClaude Codeにレビューさせろ。

```
practice/boss-challenge/forest/my-ai-employee/CLAUDE.md を読んで、以下の観点でレビューして：
1. 各セクションは具体的か？（曖昧な表現はないか）
2. 役割と禁止事項に矛盾はないか？
3. このCLAUDE.mdで実際にAIが動いたら、期待通りに振る舞うか？
4. 改善点があれば提案して
```
