## ヒント

### hint_1
悪い例 vs 良い例：

❌「ボタン作って」
✅「React + TypeScript で、青色の角丸ボタンコンポーネントを作って。
   onClick で渡された関数を呼び出すようにして」

### hint_2
効果的なプロンプトの要素：
1. 明確な目標
2. 具体的な条件
3. 期待する出力形式
4. 技術スタック情報

### hint_3
良いプロンプトのテンプレート：

「[目的]をしたい。
技術スタックは[言語/フレームワーク]。
条件は：
- [条件1]
- [条件2]
出力形式は[形式]でお願い。」

例：
「ユーザー一覧を表示するコンポーネントを作りたい。
技術スタックは React + TypeScript。
条件は：
- ユーザーの名前とメールを表示
- ローディング中は Spinner を表示
- エラー時はエラーメッセージを表示
出力形式は関数コンポーネントでお願い。」
