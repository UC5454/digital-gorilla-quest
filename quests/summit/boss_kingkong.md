## ⚔️ チャレンジ

全ての力を結集し、以下の条件を全てクリアせよ！

### 1. CLAUDE.md完成確認
AI社員のCLAUDE.mdが全セクション揃っていて、150行以上であることを確認しろ。

```
[AI社員名]/CLAUDE.md の行数を数えて、以下のセクションが全て含まれているか確認して：
- identity
- role（責任範囲、判断基準）
- persona（コミュニケーションスタイル）
- skills（業務フロースキル含む）
- startup（起動時確認事項）
- inbox_rules（連携ルール）
```

### 2. スキル動作確認
作成したスキルが正常に動作することを確認しろ。

```
/log-task "キングコング戦 - スキル動作テスト" を実行して
```

### 3. MCP接続確認
.mcp.jsonが設定され、MCP経由でのアクセスが動作することを確認しろ。

```
.mcp.json の設定内容を表示して、MCPサーバーが登録されているか確認して
```

### 4. 自律実行テスト
INBOXにタスクを送信 → AI社員が自律的に処理 → outputs/に保存 → daily-logsに記録。全行程の完了を確認しろ。

```
[AI社員名]/INBOX.md の未処理セクションに以下を追記して：

【キングコングより】最終試練だ。「あなたが学んだ全ての技術」を200字でまとめて outputs/final-answer.md に保存しろ。完了したらdaily-logsに記録しろ。

その後、start-employee.sh で起動して全行程を自動実行させて。
```

### 5. 【ボーナス】2体連携（+500BP）
2体のAI社員がINBOX連携でタスクを完了させろ。

```
AI社員AにINBOXタスクを送信 → Aが処理 → BのINBOXに結果を送信 → Bを起動 → Bが後続処理を完了
```

### 6. 【ボーナス】Agent Teams（+500BP）
Agent Teamsで2エージェント以上の並列ワークフローを実行しろ。

```
Agent Teamsで2つの調査を並列実行し、結果を統合して outputs/bonus-research.md に保存して
```
