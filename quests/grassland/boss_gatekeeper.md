## ⚔️ チャレンジ

以下の5つのタスクを全て完了せよ！

### 1. フォルダ作成
Bashを使って `practice/boss-challenge/grassland/` フォルダを作成しろ。

```
practice/boss-challenge/grassland/ フォルダを作成して
```

### 2. 戦闘記録の作成
Writeを使って `practice/boss-challenge/grassland/battle-log.md` を作成しろ。

```
practice/boss-challenge/grassland/battle-log.md を作成して、以下を書いて：

# 門番ゴリラ 戦闘記録

## 挑戦者情報
- 挑戦日: [今日の日付]
- 挑戦者: [自分の名前]

## 戦闘状況
- ステータス: 戦闘中...
```

### 3. ツールガイドの作成
Writeを使って `practice/boss-challenge/grassland/gorilla-tools-guide.md` を作成し、4つのツールの説明を書け。

```
practice/boss-challenge/grassland/gorilla-tools-guide.md を作成して、Claude Codeの4つのツール（Read, Write, Edit, Bash）について、それぞれ「何ができるか」「いつ使うか」を自分の言葉で説明して
```

### 4. 勝利宣言の追記
Editを使って `battle-log.md` に勝利宣言を追加しろ。

```
practice/boss-challenge/grassland/battle-log.md の「戦闘状況」セクションの「ステータス: 戦闘中...」を「ステータス: 勝利！」に変更して、さらに以下のセクションを追加して：

## 勝利宣言
門番ゴリラを倒した！草原の4つのツールを完全に習得！
```

### 5. 行数確認
Bashを使って `gorilla-tools-guide.md` の行数を確認しろ。

```
wc -l practice/boss-challenge/grassland/gorilla-tools-guide.md を実行して
```
