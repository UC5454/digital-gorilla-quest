## ボスチャレンジ

### 課題内容

お前が草原で学んだことを、実際にやって見せろ！

以下の3つのタスクを全て完了せよ：

### タスク1: ファイル操作の証明
`practice/boss-challenge/` フォルダに `my_first_code.js` を作成し、
以下の内容を書き込め：

```javascript
// 門番ゴリラへの挑戦状
console.log("ウホッ！俺はClaude Codeを使いこなせるゴリラだ！");
```

### タスク2: ファイル編集の証明
作成した `my_first_code.js` を編集し、2行目に以下を追加せよ：

```javascript
console.log("草原エリア制覇！次は森林だ！");
```

### タスク3: コマンド実行の証明
Claude Code を使って、作成したファイルを Node.js で実行せよ。
（`node practice/boss-challenge/my_first_code.js`）

## 成功条件

- [ ] `practice/boss-challenge/my_first_code.js` が存在する
- [ ] ファイルに2つの console.log が含まれている
- [ ] ファイルを実行して出力を確認した
