# Quest 4: バナナの木を植える

## 導入テキスト

ウホホホ！ファイルが読めるようになったか！

次は「作る」力を身につけるときだ。
ゴリラは食料を確保するため、バナナの木を植える。
お前もファイルを「植えて」みろ！

practice フォルダに何かファイルを作成するんだ。

## 目標

- Claude Codeを使って新しいファイルを作成する
- practice/ フォルダ内にファイルを作る

## コピペ

Claude Codeに以下を入力してみよう：

```
practice/banana.txt を作成して。内容は「デジタルゴリラの最初のバナナ」にして
```

## 成功条件

- [ ] practice/ フォルダ内に新しいファイルが作成された
- [ ] ファイルに何らかの内容が書き込まれている

## 報酬

- base_bp: 250
- first_clear_bonus: 50

## ヒント

### hint_1
Claude Codeに「practice フォルダに hello.txt を作成して」と頼んでみろ！
内容も一緒に指定できるぞ。

### hint_2
例えば「practice/my_first_file.md を作成して、内容は『ウホホ！これが俺の最初のファイルだ！』にして」と指示する。

### hint_3
入力例: 「practice/banana.txt というファイルを作成して。内容は『デジタルゴリラの最初のバナナ 🍌』にして」
Claude Codeが Write ツールを使ってファイルを作成してくれるぞ！
