# Quest 6: ジャングルの声を聴く

## 導入テキスト

ウホホホ！ファイル操作の基本は完璧だ！

最後に、ジャングル（システム）の声を聴く力を身につけろ。
Claude Codeはコマンドを実行することもできる。

これができれば、お前は草原の全てを把握したことになる！

## 目標

- Claude Codeを通じてシェルコマンドを実行する
- システムの情報を取得する

## コピペ

Claude Codeに以下を入力してみよう：

```
ls -la コマンドを実行して、このディレクトリの内容を見せて
```

## 成功条件

- [ ] Claude Codeにコマンドを実行させた
- [ ] 結果が表示された（例：ディレクトリ一覧、日付など）

## ヒント

### hint_1
「今のディレクトリにあるファイル一覧を表示して」と頼んでみろ！
Claude Codeが ls コマンドを実行してくれるはずだ。

### hint_2
他にも「今日の日付を教えて（dateコマンドで）」や
「現在のディレクトリのパスを教えて（pwdコマンドで）」などを試してみろ。

### hint_3
入力例:
- 「ls -la コマンドを実行して、このディレクトリの内容を詳しく見せて」
- 「date コマンドを実行して現在の日時を表示して」
- 「pwd コマンドで現在地を教えて」

Claude Codeが Bash ツールを使ってコマンドを実行してくれるぞ！
