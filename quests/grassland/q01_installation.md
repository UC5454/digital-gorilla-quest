# Quest 1: ジャングルへの第一歩

## 導入テキスト

ウホッ！若いゴリラよ、デジタルジャングルへようこそ！

お前がここに来たということは、「Claude Code」というパワーを手に入れたいのだな？
それは賢明な選択だ。この力があれば、どんなコードの密林も切り開けるぞ！

まずはその力を召喚するところから始めようか。

## 目標

- Claude Codeをインストールする
- バージョンが正しく表示されることを確認する

## 成功条件

- [ ] `claude --version` コマンドが正常に動作する
- [ ] バージョン番号が表示される

## ヒント

### hint_1
Claude Codeは npm からインストールできるぞ。
`npm install -g @anthropic-ai/claude-code` を試してみろ！

### hint_2
インストールが終わったら、ターミナルで `claude --version` と入力するんだ。
バージョン番号が表示されれば成功だ！

### hint_3
もし npm がないなら、まずは Node.js をインストールする必要があるぞ。
https://nodejs.org/ からダウンロードできる。
インストール後、`npm install -g @anthropic-ai/claude-code` でOKだ！
