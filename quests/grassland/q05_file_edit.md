# Quest 5: 縄張りの整備

## 導入テキスト

ウホッ！ファイルを作れるようになったな！

だが、ゴリラの縄張りは作るだけでは終わらない。
常に整備し、より良くしていく必要がある。

さっき作ったファイルを編集して、内容を追加してみろ！

## 目標

- 既存のファイルを編集する
- 内容を追加または変更する

## コピペ

Claude Codeに以下を入力してみよう：

```
practice/banana.txt の末尾に「ゴリラは今日も元気だ！ウホホ！」を追加して
```

## 成功条件

- [ ] Quest 4で作成したファイル（または別のファイル）を編集した
- [ ] 元の内容に何かが追加されている

## 報酬

- base_bp: 250
- first_clear_bonus: 50

## ヒント

### hint_1
Claude Codeに「practice/banana.txt に新しい行を追加して」と頼んでみろ！
何を追加するか具体的に言うといいぞ。

### hint_2
「practice/banana.txt を編集して、末尾に『追加のバナナ 🍌🍌』という行を追加して」
というように指示してみろ。

### hint_3
入力例: 「practice/banana.txt を開いて、ファイルの最後に改行して『ゴリラは今日も元気だ！ウホホ！』を追加して」
Claude Codeが Edit ツールを使って編集してくれるぞ！
