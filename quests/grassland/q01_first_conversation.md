# Quest 1: 最初のウホウホ会話

## 🦍 導入

ウホッ！ようこそ、若きゴリラよ！
ここは広大な草原——全てのゴリラの旅が始まる場所だ。

お前の前には「Claude Code」という強力な相棒がいる。
だがな、こいつは普通のチャットAIとはワケが違う。
まずは話しかけて、その力を確かめてみろ！

## 🎯 目標

- [ ] Claude Codeに話しかけて対話できることを確認する
- [ ] Claude Codeが普通のチャットAIと何が違うか理解する
- [ ] Escape / Shift+Tab の操作を確認する

## 📋 やること

1. Claude Codeに以下のコピペを送信する
2. 返ってきた回答を読んで、Claude Codeの特徴を理解する
3. Escapeキーを押すと実行をキャンセルできることを確認する
4. Shift+Tabでモード切替ができることを確認する

## 💡 ヒント

### hint_1
ターミナルで `claude` と入力すればClaude Codeが起動するぞ。あとは普通に日本語で話しかけるだけだ！

### hint_2
Claude Codeは「ターミナルで動くAI開発者」だ。チャットだけじゃなく、ファイルの読み書き・編集・コマンド実行の4つのツールを自動で使い分ける。そこが普通のAIチャットとの決定的な違いだ。

### hint_3
起動: `claude`
入力: 「自己紹介して。あと、あなたは普通のチャットAIと何が違うの？」
操作確認: Escapeで中断、Shift+Tabでモード切替
終了: `/exit` または Ctrl+C

## 📚 学習ポイント

- Claude Codeは「ターミナルで動くAI開発者」。チャットAIとは根本的に違う
- 4つのツール（Read / Write / Edit / Bash）を自動で使い分ける
- Escapeで実行キャンセル、Shift+Tabでモード切替ができる
