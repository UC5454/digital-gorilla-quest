# Quest 15: 最初の技を編み出す

## 導入テキスト

ウホホ！技の型を理解したな！

いよいよ、お前自身の技を編み出すときだ。
practice/skill-dojo/ フォルダに、最初のスキルを作成しろ！

シンプルな技でいい。まずは作ることが大切だ。

## 目標

- 自分のスキルを作成する
- 実際に呼び出して動作を確認する

## 成功条件

- [ ] practice/skill-dojo/.claude/skills/[name]/SKILL.md を作成した
- [ ] スキルの基本構造（タイトル、起動時の動作）が含まれている

## 報酬

- base_bp: 500
- first_clear_bonus: 100

## ヒント

### hint_1
まずディレクトリ構造を作るところから。
practice/skill-dojo/.claude/skills/hello/ を作成して、
その中に SKILL.md を置くんだ。

### hint_2
シンプルな「挨拶スキル」を作ってみろ：

```markdown
# /hello - 挨拶する

ユーザーに元気な挨拶をする。

## 起動時の動作

1. ユーザーの名前を確認（あれば）
2. 元気な挨拶メッセージを表示
3. 今日の日付も添える

## 出力形式

🌟 こんにちは、[名前]さん！
今日は[日付]です。
良い一日を！
```

### hint_3
Claude Code に以下を頼む：

「practice/skill-dojo/.claude/skills/hello/SKILL.md を作成して。
内容は：

# /hello - 元気に挨拶する

ユーザーに元気な挨拶をするスキル。

## 起動時の動作

1. 現在の時刻を確認
2. 時間帯に応じた挨拶を選択
   - 朝（5-11時）: おはようございます
   - 昼（12-17時）: こんにちは
   - 夜（18-4時）: こんばんは
3. 挨拶メッセージを表示

## 出力形式

[挨拶]！今は[時刻]です。
」
