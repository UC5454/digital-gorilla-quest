「スキル」とは何か？
それは、繰り返し使う操作を「/コマンド名」で一発呼び出しできるようにしたものだ。
例えば `/status` と打つだけで、プロジェクトの状態を一覧表示できたりする。

## 💡 ヒント

### hint_1
スキルは `.claude/commands/` フォルダに Markdown ファイルとして保存されるぞ。ファイル名がそのままコマンド名になる。

### hint_2
スキルの基本構造:
```
.claude/
└── commands/
    └── greet.md     → /greet で呼び出せる
    └── status.md    → /status で呼び出せる
```
各 `.md` ファイルの中に、AIへの指示が書かれている。

### hint_3
コピペをそのまま送信すればOK。Claude Codeが `.claude/commands/` を調べて、既存スキルの構造を解説してくれる。ポイントは「スキル = Markdownファイルに書いた指示を `/コマンド名` で呼び出せる仕組み」だということ。

## 📚 学習ポイント

- **スキル = Custom Slash Commands**: `/コマンド名` で呼び出せるカスタム命令
- **格納場所**: `.claude/commands/` フォルダにMarkdownファイルとして保存
- **ファイル名 = コマンド名**: `greet.md` → `/greet` で呼び出せる
