## ⚔️ チャレンジ

2体のAI社員が連携してタスクを完了するデモンストレーションを行え！

### 1. フォルダ準備
`practice/boss-challenge/fortress/` フォルダを作成しろ。

```
practice/boss-challenge/fortress/ フォルダを作成して
```

### 2. AI社員Aにタスクを指示
AI社員Aに「ゴリラの生態について3つの面白い豆知識」を調べさせ、`practice/boss-challenge/fortress/gorilla-facts.md` に保存させろ。

```
[AI社員Aの名前]として、ゴリラの面白い豆知識を3つ調べて practice/boss-challenge/fortress/gorilla-facts.md に保存して。完了したら [AI社員Bの名前]/INBOX.md に「gorilla-facts.mdを作成しました。読みやすく編集してください」と書いて。
```

### 3. AI社員Bを起動して後続処理
AI社員Bを起動し、INBOXのメッセージに基づいてgorilla-facts.mdを編集・改善させろ。

```
[AI社員Bの名前]として、INBOX.mdを確認して未処理メッセージを処理して。
```

### 4. daily-logsに記録
両方のAI社員の活動をdaily-logsに記録しろ。

```
my-first-ai/daily-logs/ に今日の日付のファイルを作成して、AI社員Aの作業内容とAI社員Bの作業内容を両方記録して。
```
