## ⚔️ チャレンジ

3エージェント以上の並列・直列混合ワークフローを構築し、最終成果物を完成させよ！

### 1. チーム編成
「volcano-boss」チームを作成し、以下のタスクを登録しろ。

### 2. タスク設計（並列+直列の混合DAG）

```
Phase 1（並列）:
  - タスクA: 「ゴリラの強さ」について3つのポイントを調べる
  - タスクB: 「ゴリラの優しさ」について3つのポイントを調べる
  - タスクC: 「ゴリラの知恵」について3つのポイントを調べる

Phase 2（直列 — Phase 1全完了後）:
  - タスクD: 3つの調査結果を統合して「最強のゴリラ図鑑」を作成する

Phase 3（直列 — Phase 2完了後）:
  - タスクE: 図鑑をレビューし、最終版を完成させる
```

### 3. 実行

```
以下のワークフローをAgent Teamsで実行して：

  A: 「ゴリラの強さ」を3ポイントで調査（Explore）

  C: 「ゴリラの知恵」を3ポイントで調査（Explore）

  D: A+B+Cの結果を統合して「最強のゴリラ図鑑」を作成（general-purpose）

  E: レビュー＆最終版を practice/boss-challenge/volcano/gorilla-encyclopedia.md に保存（general-purpose）
```
