# 🦍 Digital Gorilla Quest - ジャングルマスター設定 🦍

## 世界観

ここは広大な「デジタルジャングル」。
あなたは株式会社デジタルゴリラの見習いゴリラとして、Claude Codeの修行を積み、
最強のシルバーバック、そして伝説の「ジャングルの王」を目指す冒険に出る。

5つのテリトリーを制覇し、各エリアのボスを倒して頂点を目指せ！

## コマンド一覧

### メインコマンド（これだけ覚えればOK！）

`/go` - 全部入りの統合コマンド

| 入力 | 動作 |
|------|------|
| `/go` | 次のクエストを表示（初回はゲーム開始） |
| `/go done` or 「できた」 | クエスト完了 → BP付与 → 自動で次のクエスト表示 |
| `/go hint` | 段階的ヒント表示（3段階） |
| `/go status` | コンパクトなステータス表示 |
| `/go boss` | ボス戦開始 |
| `/go [番号]` | 指定クエストに移動 |

### レガシーコマンド（引き続き使用可能）

- `/jungle` - ゲーム開始・進行案内（ジャングルマスター登場）
- `/quest [番号]` - クエスト開始（冒険スタート）
- `/banana` - クエスト完了判定（バナナポイント獲得）
- `/status` - ステータス確認（ゴリラの雄叫び）
- `/hint` - ヒント表示（先輩ゴリラの助言）
- `/boss` - ボス戦開始（ドラミング演出）

## ゲームルール

1. **クエストをクリアしてBP（バナナポイント）を獲得**
2. **BPが貯まるとパワーレベルが上がる**
3. **パワーレベルに応じて称号が進化**
4. **各エリアの全クエストをクリアするとボス戦解禁**
5. **ボスを倒すと次のエリアへ進める**
6. **「できた」と言えば自動で次のクエストに進む（`/go` 使用時）**

## 進捗データ

ゴリラの進捗は `gorilla_data/progress.json` に保存される。
ローカル設定は `CLAUDE.local.md` に記録。
